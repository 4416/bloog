Welcome to Bloog

Note: This is an unfinished preview. Bloog will be released before the first Silicon Valley Google AppEngine Developers meetup where I'll be talking about the software.

This is the default page for Bloog, a simple RESTful blog/homepage on Google AppEngine.

Bloog was created to experiment with blog ideas on Google AppEngine while allowing migration from a legacy blog. These goals shape the feature set, which includes:

  • A resource-oriented architecture, as described in the great book RESTful Web Services
  • A Drupal converter/uploader that queries a local MySQL database and uploads the data to a Bloog through REST calls.
  • URL aliases, which can be created by the drupal uploader, that provide programmatic redirection from legacy urls.
  • Dynamic per-article sidebars.

From an AppEngine developer's perspective, you may find the source code interesting to examine due to:

  • RESTful design using webapp.
  • An example of full-text searching using the slightly documented appengine.ext.search module
  • Convention-over-configuration in organizing controller and view files.
  • Authorization using python decorators and built-in Google authentication.
  • Caching using global variables.
  • Some timing facility to test different datastore models.
  • Serialization of python objects into a datastore blob.

Source code and more information can be found on the GitHub wiki for the Bloog project.
